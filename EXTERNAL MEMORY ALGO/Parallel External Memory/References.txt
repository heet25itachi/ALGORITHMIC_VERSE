## References ##

1. Wikipedia Article: Parallel External Memory

URL: https://en.wikipedia.org/wiki/Parallel_external_memory
Description: The primary source for the PEM model and PEMDISTSORT algorithm, describing the model’s structure with $ P $ processors, private caches of size $ M $, blocks of size $ B $, and shared main memory of size $ N $. It details PEMDISTSORT’s multiway partitioning, pivot selection, and I/O complexity $ O\left(\frac{N}{PB}\log_{M/B}\frac{N}{B}\right) $ under constraints $ P \leq \frac{N}{B^2} $, $ M = B^{O(1)} $. The pseudocode implements this approach, using $ N = 16 $, $ P = 4 $, $ M = 8 $, $ B = 2 $, $ d = 4 $, and the test array from the prior artifact.
